| 模块    | 子模块      | 功能描述                          |
|-------|----------|-------------------------------|
| 登录    | 账号密码     | 输入账号密码进行登录                    |
|       | 密码隐藏     | 输入密码的时候可以选择隐藏输入               |
|       | 手机号验证码   | 填写手机号，发送验证码，验证通过后登录           |
| 搜索    | 文字搜索     | 输入运单号/手机号/收件人/收货地址等进行搜索       |
|       | 扫码搜索     | 扫描条形码进行搜索                     |
| 消息    | 消息列表     | 展示消息列表                        |
|       | 消息分类     | 消息分为公告、系统通知（寄件、派件、快件取消、签收提醒等） |
|       | 日期筛选     | 通过近一个月的时间段进行筛选消息              |
|       | 全部已读     | 对未读消息列表进行一键已读操作               |
| 取件    | 搜索       | 输入内容搜索相应取件内容                  |
|       | 分类       | 取件任务分为待取件/今日已取                |
|       | 筛选       | 通过对快件的距离/时间/超时任务进行筛选          |
|       | 信息加密     | 用户端下单勾选信息加密，快递员端对姓名/手机号部分加密显示 |
|       | 取消取件     | 选择取消原因进行取消取件操作                |
|       | 编辑资料     | 完善用户寄件信息，根据托寄物实际情况计算运费        |
|       | 列表管理     | 取件列表可进行全选、转单、删除、批量打印等操作       |
|       | 转单       | 待取件任务可进行转单操作                  |
|       | 订单跟踪     | 查看物流跟踪记录、到达网点记录及时间等           |
|       | 复制单号     | 点击可复制运单号                      |
|       | 运费编辑     | 对取件订单进行运费修改或编辑                |
|       | 物品分类     | 对取件物品进行品类选择                   |
|       | 备注       | 对用户备注进行修改及自定义备注               |
|       | 超时任务     | 对取件任务进行时间判断，显示超时提醒            |
|       | 取件       | 对用户下的订单进行确认取件操作               |
| 派件    | 搜索       | 输入内容搜索相应派件内容                  |
|       | 分类       | 派件任务分为待派送/已签收                 |
|       | 筛选       | 通过快件距离/时间/即将超时进行筛选            |
|       | 呼叫用户     | 给收方用户打电话                      |
|       | 发信息      | 给收方用户发信息                      |
|       | 订单跟踪     | 查看物流跟踪及到达记录                   |
|       | 复制单号     | 可复制运单号                        |
|       | 签收       | 用户手写签字收件                      |
|       | 拒收       | 用户拒绝收件处理                      |
| 订单跟踪  | 地图轨迹     | 在地图上显示派送地址的轨迹线路               |
| 历史取派  | 文字搜索     | 支持文字搜索或扫码搜索历史订单               |
|       | 日期筛选     | 通过近一个月的时间段进行历史订单筛选            |
|       | 删除       | 删除历史快件订单记录                    |
|       | 筛选分类     | 可选择近一个月的历史取派任务（含今日及未来两天）      |
|       | 状态标签     | 对任务增加相应的状态标识标签                |
| 收款/支付 | 收款       | 对未支付任务显示支付状态，并可发起收款           |
|       | 支付兼容技术   | 前端调取后端链接，生成支付二维码              |
|       | 支付       | 写死，自动支付成功                     |
| 运费查询  | 地址选择     | 选择寄件与派件地址以计算运费                |
|       | 体积选择     | 选择货物的体积大小                     |
|       | 重量选择     | 选择货物的重量大小                     |
|       | 查询       | 点击查询相应货物的预估费用                 |
| 个人中心  | 头像/姓名/账号 | 显示系统信息（不可编辑）                  |
|       | 更改手机号    | 更改绑定的手机号                      |
|       | 我的排班     | 显示快递员的排班时间范围                  |
|       | 所属网点     | 显示快递员所属的网点信息                  |
